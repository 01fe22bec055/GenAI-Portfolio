Course Project: SDXL Image Generator

This project is a customized image generator built using Stable Diffusion XL (SDXL Base 1.0). It allows users to input text prompts and get stylistically personalized images. Training involved LoRA and DreamBooth techniques on a local dataset. Frontend was integrated using Gradio and evaluation tracked using Weights & Biases.

Other Projects

Road Damage Detection using YOLO

Developed a YOLOv5-based system to detect and classify road damage types like potholes, cracks, and rutting using real-world datasets. It can assist in predictive maintenance and smart city infrastructure planning.

3D Avatar Deformation using SMPL-X

Built a simulation pipeline that uses the SMPL-X model to create natural human avatar movements by interpreting body pose and shape. Integrated it with Blender for real-time rendering.

3D Model Relighting using Transformers

Created a transformer-based solution for dynamically relighting 3D models under different virtual environments. Improves realism for games and virtual production.
